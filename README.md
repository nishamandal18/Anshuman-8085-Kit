# Anshuman 8085 Kit Simulator
![Anshuman Kit](image_2d7ec2.jpg)

A fully functional web-based simulator for the Anshuman 8085 Microprocessor Trainer Kit. This tool allows students and hobbyists to write, execute, and debug 8085 assembly code directly in the browser.

üöÄ Features

Realistic Interface: Modeled after the actual Anshuman hardware kit.

7-Segment Display: Visualizes address, data, and system messages.

Full Keypad Support: Input hex codes and commands just like the real device.

Core Instructions: Supports MOV, MVI, ADD, STA, IN, OUT, and more.

Input/Output Ports: Dedicated LED visuals for Output Ports 01 & 02.

üéÆ How to Use

1. Enter a Program

Press [ESC] to Reset.

Press [S] -> [CR] -> [CR] to enter Substitute Memory mode.

Type the starting address (e.g., 7000) and press [CR].

Enter your Opcodes one by one, pressing [CR] after each.

2. Execute

Press [ESC] to exit entry mode.

Press [G] -> [CR] -> [CR].

Type the starting address (e.g., 7000) and press [CR].

3. Check Results

Registers: Press [S] -> [CR] -> [A] (or B, C, D) to view register values.

Memory: Press [S] -> [CR] -> [CR] -> Enter Address -> [CR].

Output Ports: Watch the LED indicators at the bottom of the kit.
